# Zomato-Restaurants-Analysis

Zomato Restaurant Analytics Dashboard (Power BI + SQL)

This project analyzes global restaurant trends using the Zomato dataset.
It includes key insights on city performance, cuisine popularity, pricing patterns, and ratings.

Tools Used
	•	Power BI
	•	MySQL
	•	DAX
	•	Data Modeling

Dashboard Features
	•	KPI section (Total Restaurants, Cities, Avg Rating, Unique Cuisines)
	•	Top cities by restaurant count
	•	Cuisine popularity
	•	Rating distribution
	•	Average cost by city
	•	Interactive slicers (City, Cuisine, Rating Category, Price Range)
